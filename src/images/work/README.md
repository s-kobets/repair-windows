# Work Icons Directory

This directory contains work process icons for the website.

Images should be downloaded using:
```bash
npm run download-images
```

Or using the bash script:
```bash
./scripts/download-images.sh
```

Expected files:
- icon-1.png (Заявка)
- icon-2.png (Менеджер)
- icon-3.png (Договор и исполнение)
- icon-4.jpg (Оплата)
